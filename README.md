# Selfbot

This selfbot is made for educational purpose only.

## Requirements

[NodeJS](https://nodejs.org/) to run the selfbot.
Then in the selfbot directory from CMD:
```bash
npm i
```

## Usage

```bash
node index.js
```
Or you can start "Start.bat"
## Features
• DM ALL

• Fake Token Grabber

• User Info

• Send a crash video to someone

• and MORE
